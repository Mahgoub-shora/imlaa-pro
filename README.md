# 📚 إملاء برو – Imlaa Pro

تطبيق تعليمي تفاعلي يساعد الأطفال على تعلم الإملاء العربي بطريقة ممتعة وذكية باستخدام أساليب حديثة تعتمد على اللعب والتحفيز.

---

## 🚀 Live Demo

🔗 https://imlaa-pro.pages.dev

---

## 🎯 الفكرة

إملاء برو هو تطبيق بسيط وسريع يهدف إلى:

* تحسين مهارات الكتابة والإملاء
* تدريب الأطفال باستخدام التكرار الذكي
* تحويل التعلم إلى تجربة ممتعة 🎮

---

## ✨ المميزات

* 👁️ نمط بصري (عرض الكلمة)
* 🎧 نمط سمعي (استمع واكتب)
* 🧩 نمط الحرف الناقص
* 🏆 نظام نقاط وتحفيز
* 📊 تقارير أداء ورسوم بيانية
* 🔁 إعادة الكلمات الصعبة تلقائياً
* 📱 يعمل على جميع الأجهزة

---

## 💳 نموذج الربح (SaaS)

* نسخة مجانية (محدودة)
* نسخة مدفوعة عبر Stripe
* نظام تفعيل باستخدام كود اشتراك

---

## 🛠️ التقنيات المستخدمة

* HTML / CSS / JavaScript
* Chart.js (للرسوم البيانية)
* LocalStorage
* Stripe (الدفع)
* Cloudflare Pages (الاستضافة)
